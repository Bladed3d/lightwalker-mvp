Technical Implementation Specifications

Animation Specifications:

  • Timeline Auto-scroll: Smooth 1px/minute movement, CSS transform: translateX()
  • Hero Activity Pulse: 2s infinite pulse animation on current activity icon
  • Container Shimmer: 3s infinite shimmer effect on active containers
  • Status Indicators: 1s blink animation for current activities
  • Navigation Transitions: 0.3s ease-in-out for timeline position changes

Interactive Elements:

  • Drag Timeline: Mouse/touch drag with momentum scrolling
  • Arrow Navigation: 30-minute jumps forward/backward
  • NOW Button: Instant snap to current time with smooth animation
  • Activity Hover: Expand cards showing detailed guidance
  • Container Switching: Click to activate different role model containers

Responsive Breakpoints:

  • Desktop (1200px+): Full container system side-by-side
  • Tablet (768px-1199px): Stacked containers, compressed timeline
  • Mobile (< 768px): Single column, simplified timeline icons
