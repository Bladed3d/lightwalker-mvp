## Key Wireframe Principles

### 1. Information Hierarchy
- **Primary**: Current activity always visible and prominent
- **Secondary**: Timeline shows daily progression
- **Tertiary**: Quick actions and role model influences support main flow

### 2. Progressive Disclosure
- Timeline items collapse/expand for details
- Role model container shows influences without overwhelming
- Quick actions provide immediate access without cluttering

### 3. Responsive Behavior
- **Mobile**: Vertical stacking, single column, finger-friendly targets
- **Desktop**: Multi-column layout, hover states, keyboard navigation
- **Tablet**: Hybrid approach with collapsible panels

### 4. Visual Flow
- Top-to-bottom reading pattern
- Current activity draws immediate attention
- Timeline creates natural progression sense
- Quick actions positioned for easy thumb access (mobile)

### 5. Gamification Integration
- Progress bars provide immediate feedback
- Discovery Points visible throughout interface
- Role model influences show behavioral context
- Achievement states clearly differentiated (done/next/soon/planned)
